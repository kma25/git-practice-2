\## Введение



Git — это распределенная система контроля версий. Она позволяет сохранять историю изменений проекта, работать с ветками и возвращаться к предыдущим версиям файлов.



GitHub — это онлайн-сервис для хранения Git-репозиториев и совместной работы над проектами.



\---



\## Создание учетной записи GitHub



Для создания учетной записи GitHub необходимо:



1\. Перейти на сайт GitHub.

2\. Нажать кнопку `Sign up`.

3\. Ввести адрес электронной почты.

4\. Придумать пароль.

5\. Указать имя пользователя.

6\. Подтвердить регистрацию через письмо на email.

7\. После входа в аккаунт создать новый репозиторий.



Пример названия репозитория:



```text

git-instruction

```

\## Установка Git



Для установки Git необходимо:



1\. Перейти на официальный сайт Git.

2\. Скачать установщик для Windows.

3\. Запустить установщик.

4\. На этапах установки оставить стандартные параметры.

5\. После установки открыть Git Bash.



Проверка установки Git:



```bash

git --version

```



Пример результата:



```text

git version 2.45.0

```

\## Итоговая схема работы



Итоговый вариант инструкции получается после слияния веток `local` и `global` в основную ветку `main`.



```text

main   — структура документа, создание GitHub и установка Git

local  — команды для локальной работы с Git

global — команды для удаленного Git и форков

```
